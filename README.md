# 📌 Project Description: AI Job Search Assistant 🇮🇳

The **AI Job Search Assistant** is a smart career development platform designed to simplify and automate the job search process for students and professionals, with a special focus on the Indian job market. The system integrates job search functionalities, resume analysis, and company insights into a single, user-friendly interface.

This project leverages modern AI techniques and data-driven approaches to help users discover relevant job opportunities, optimize their resumes for Applicant Tracking Systems (ATS), and make informed career decisions.

---

# 🎯 Objective

The main objective of this project is to:

* Automate job searching based on user preferences
* Analyze and improve resumes for better ATS compatibility
* Provide personalized job recommendations
* Offer insights into companies, salaries, and job trends

---

# ⚙️ Key Features

### 🔹 Job Search & Filtering

* Search jobs based on:

  * Location (e.g., Bangalore, Hyderabad)
  * Salary (in LPA – Lakhs Per Annum)
  * Work-from-home options
* Display relevant job listings with filters and sorting

---

### 🔹 Resume Analysis

* Upload resume (PDF format)
* Extract:

  * Skills
  * Education
  * Experience
* Identify missing skills based on job descriptions

---

### 🔹 Job Matching System

* Match user resume with job descriptions
* Generate a **match score (%)**
* Suggest improvements for better alignment

---

### 🔹 ATS Optimization

* Provide feedback to improve resume quality
* Suggest:

  * Keywords
  * Action verbs
  * Formatting improvements

---

### 🔹 Company Insights

* Display:

  * Company ratings
  * Salary ranges
  * Basic culture insights

---

### 🔹 User Interface

* Built using Streamlit for simplicity and speed
* Clean and interactive dashboard for job seekers

---

# 🧠 Technologies Used

* **Frontend:** Streamlit
* **Backend:** Python
* **Libraries:**

  * pdfplumber (resume parsing)
  * scikit-learn (job matching)
* **AI Tools:** Basic NLP / LangChain (optional)
* **Database:** SQLite (for saving job data)

---

# 🌍 Indian Market Focus

This project is tailored for the Indian job ecosystem with features like:

* Salary representation in **LPA (Lakhs per annum)**
* Support for metro and Tier-2 city job searches
* Work-from-home filtering
* Entry-level and fresher job support
